## Pourquoi ce guide existe

On a construit Symbad — un assistant IA complet — depuis un smartphone Android,
dans Chrome, en WiFi. Sans PC. Sans clavier. Sans souris.

On a documenté chaque bug, chaque solution, chaque règle.
Ce dépôt est le guide qu'on aurait aimé trouver au début.
